# ✦ Alma Connection

> "O que aconteceria se o Tinder encontrasse a premissa de 'Casamento às Cegas' da Netflix?"

O **Alma** é um projeto de portfólio desenvolvido para desafiar a superficialidade das conexões modernas, focando na personalidade e afinidades intelectuais antes da imagem.

## 🚀 Tech Stack
- **Frontend:** React + Vite (UI focada em luxo e minimalismo)
- **Backend:** Java Spring Boot (Segurança com JWT e BCrypt)
- **Banco de Dados:** PostgreSQL
- **Ambiente:** Desenvolvido integralmente em Ubuntu Linux

## 🛠️ Funcionalidades
- Fluxo de Onboarding progressivo para maior retenção.
- Algoritmo de afinidade baseado em interesses culturais.
- Interface Dark Mode "Premium" com tipografia clássica.

## 💼 Portfólio
Este projeto demonstra competências em:
- Arquitetura Full Stack.
- Design System e UX/UI refinado.
- Criptografia e proteção de dados (AES-256).
